Program:
RecitingWord

Function of the program:
Allow the users to recite more than 5000 SAT vocabularies, by using the scientific memory curve.

Instruction:
Click "Start" or enter any keys to start the program
Click "I'm Noob" or enter "B" for unfamiliar vocabulary
	--Then click "Next" or enter any key for the next word
Click "Know" or enter "K" for knowing this vocabulary
	--Click "Correct (C)" or "Incorrect (I)"
Wiping the memory for this time or all is optional
